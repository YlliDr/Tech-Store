Component Examples

See how to use all the reusable components in your pages

Product Card Component

The ProductCard component displays product information in a beautiful card format. It includes quick add to cart functionality and emits custom events. 

<product-card 
  name="Gaming Laptop Pro" 
  price="1299.00"
  image="laptop.jpg"
  rating="5"
  reviews="156"
  link="product.html">
</product-card>

Review Card Component

Display customer reviews with ratings, author information, and review content. Perfect for product pages and testimonials. 

<review-card 
  rating="5" 
  title="Excellent product!"
  author="John Doe"
  date="15.11.2024"
  content="This product exceeded my expectations...">
</review-card>

Image Gallery Component

An interactive image gallery with thumbnail navigation. Perfect for product pages to showcase multiple product images. 

<image-gallery 
  images="image1.jpg,image2.jpg,image3.jpg">
</image-gallery>

Toast Notification Component

Show beautiful toast notifications with different types and auto-dismiss. Try clicking the buttons below! 

// Show a success notification
showToast('Product added to cart!', 'success');

// Show an error notification
showToast('Something went wrong!', 'error');

// Show a warning notification
showToast('Please check your input', 'warning');

// Show an info notification
showToast('New feature available!', 'info');

Event Handling

Components can emit custom events that you can listen to. Try clicking the "Quick Add" button on any product card above! 

// Listen for product-added events
document.addEventListener('product-added', (event) => {
  const { productName, productPrice } = event.detail;
  console.log(\`Added: \${productName} - \${productPrice}€\`);
  
  // Add to cart, show notification, etc.
  updateCart(productName, productPrice);
});

Component Props Reference

ProductCard Props:

Prop Type Default Description
name String "Product Name" Product name/title
price String "0.00" Product price
image String (URL) Placeholder Product image URL
rating Number (1-5) 5 Star rating
reviews String "0" Number of reviews
link String (URL) "#" Product detail page link

ReviewCard Props:

Prop Type Default Description
rating String "5" Star rating (1-5)
title String "Bewertung" Review title
author String "Anonym" Reviewer name
date String Current date Review date
content String "" Review text
